# Titanic Survival Analysis using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand passenger demographics, survival patterns, and important factors influencing survival.

## Dataset
Titanic Dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

## Key Findings

- Most passengers travelled in Third Class.
- Female passengers had a higher survival rate.
- Cabin contained significant missing values.
- Passenger Class influenced survival.
